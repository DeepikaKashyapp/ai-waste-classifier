
  # AI Waste Classifier 

  This is a code bundle for AI Waste Classifier . The original project is available at https://ephemeral-duckanoo-f4aaee.netlify.app/

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  ## Running the ML Backend
  
  Run `python app.py` inside the `wasteml` folder to start the YOLOv8 Flask server.
